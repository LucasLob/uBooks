# uBooks
uBooks se encarga del almacenado de información sobre libros y venta de libros y eBooks. Esta primera versión incorpora un CRUD hecho con PHP,Javascript,jQuery y Ajax. Uso de sesiones. Control de usuarios:si es administrador ve una página donde puede editar, borrar y dar de alta nuevos libros, en caso contrario solo puede consultar los libros existentes el ba base de datos y acceder a una API (Google Books) y buscar libros por autor o por título.
